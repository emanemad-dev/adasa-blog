## Project Overview {#project-overview}

**عدسة** هي منصة مدونة تصوير فوتوغرافي حديثة باللغة العربية، تركز على:

- 🎓 مقالات تعليمية عن تقنيات التصوير، نصائح الإضاءة، البورتريه، والمناظر الطبيعية
- 🛒 مراجعات معدات التصوير الاحترافية
- 🔍 تصفح مرن بأوضاع الشبكة والقائمة مع دعم RTL كامل للعربية
- 🌙 تصميم داكن عصري مع لمسات برتقالية (#ff7a00) وتأثيرات glassmorphism

مبنية بـ React + Vite لأداء فائق السرعة، مع بيانات ثابتة في `blogs.json` (25+ مقالة).

## ✨ Key Features {#key-features}

### ✅ Responsive Design

- متجاوبة كاملاً عبر جميع الأجهزة (موبايل، تابلت، سطح مكتب)
- breakpoints: 380px, 480px, 576px, 768px, 992px, 1200px

### 🔍 Article Filtering & Search

- فلترة بالفئات: إضاءة، بورتريه، مناظر طبيعية، تقنيات، معدات
- بحث فوري في العناوين والمحتوى
- تبديل عرض الشبكة/القائمة

### 📄 Article Details Page

```
┌─────────────────────────────────────┐
│ Hero Image + Metadata + Reading Time│
├─────────────────────────────────────┤
│ Table of Contents (Smooth Scroll)   │
│ Article Content                     │
│ Interactive Tags                    │
│ Social Share (Twitter/FB/LinkedIn)  │
│ Author Bio + Avatar                 │
│ Related Articles                    │
│ Newsletter (localStorage persistent)│
└─────────────────────────────────────┘
```

### 🎯 Other Features

- 📌 Sticky Navigation مع smooth scrolling
- 🏷️ نظام tags تفاعلي مع ألوان
- ✨ Animations: fade-in, slide-up, hover effects
- ♿ Accessibility: Keyboard nav, semantic HTML, focus states
- 🌐 RTL support كامل للعربية (Tajawal font)
- 📰 Newsletter signup مع localStorage
- 🔥 Featured/Selected articles section

## 🛠️ Tech Stack {#tech-stack}

| Category     | Technologies                                            |
| ------------ | ------------------------------------------------------- |
| **Frontend** | React 19 (Hooks: useState/useEffect/useRef/useNavigate) |
| **Routing**  | React Router DOM v7                                     |
| **Styling**  | Custom CSS + CSS Vars + Bootstrap 5 + TailwindCSS 4     |
| **Icons**    | React Icons (Fa/Fi/Io/Bi/Tb) + Lucide React             |
| **Build**    | Vite 7 + PostCSS/Autoprefixer                           |
| **Data**     | Static JSON (blogs.json) + localStorage                 |
| **Fonts**    | Tajawal (Arabic) + Segoe UI fallback                    |

```bash
npm create vite@latest adasa-blog -- --template react
npm i react-router-dom bootstrap react-icons lucide-react
npm i -D tailwindcss postcss autoprefixer
```

## 📁 Project Structure {#project-structure}

```
adasa-blog/
├── public/           # Static assets

└── vite.config.js    # Vite config
```

## 🎨 Design System {#design-system}

### Colors

```css
:root {
  --primary: #ff7a00; /* Orange accent */
  --bg-dark: #0f0f0f;
  --bg-card: #111;
  --bg-hover: #1a1a1a;
  --text: #f5f5f5;
}
```

### Typography

- **Arabic**: Tajawal (300/400/500/700/800)
- **Fallback**: Segoe UI

### Components

- Cards, badges, pills, gradient BGs
- Glassmorphism effects
- Animations: fade-in (GSAP-like), pulse, shake, rotate

## 📱 Responsive Features {#responsive-features}

- **Mobile-first** approach
- Horizontal scroll for categories (mobile)
- Collapsible mobile navbar
- Grid: 5-col desktop → 1-col mobile
- Sticky filter bar + scroll-to-top
- Touch-friendly interactions

## 🌐 Browser Support {#browser-support}

- Chrome 90+, Firefox 88+, Safari 15+, Edge 90+
- ✅ RTL Arabic support
- ✅ Touch devices optimized

## 🚀 Quick Start {#quick-start}

```bash
# Clone & Install
git clone <repo> adasa-blog
cd adasa-blog
npm install

# Development
npm run dev

# Build & Preview
npm run build
npm run preview
```

**Live Demo**: Open `index.html` in browser or `npm run dev` → http://localhost:5173
